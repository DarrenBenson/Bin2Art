# Support

## Getting Help

If you need help with Bin2Art:

1. **Check the README** - [README.md](README.md) covers installation, usage, and examples
2. **Search existing issues** - Your question may already be answered in [GitHub Issues](https://github.com/DarrenBenson/Bin2Art/issues)
3. **Open an issue** - If you can't find an answer, open a new issue with:
   - What you're trying to do
   - What you expected to happen
   - What actually happened
   - Your Python version and operating system

## Reporting Bugs

Use the [bug report template](https://github.com/DarrenBenson/Bin2Art/issues/new?template=bug_report.md) to report issues.
